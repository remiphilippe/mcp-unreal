# UBlueprintGeneratedClass

**Parent**: UClass
**Module**: Engine

UBlueprintGeneratedClass is the UClass subclass generated by compiling a Blueprint asset. It contains the bytecode for Blueprint graphs, default object values, and component templates. Understanding this class is essential for programmatic Blueprint manipulation — querying graphs, adding nodes, and modifying variables.

## Key Properties

- `SimpleConstructionScript` — The USimpleConstructionScript defining the component hierarchy (SCS)
- `UbergraphPages` — Array of UEdGraph representing event graph pages
- `FunctionGraphs` — Array of UEdGraph representing custom function graphs
- `DelegateSignatureGraphs` — Graphs for custom event dispatchers
- `ComponentTemplates` — Array of UActorComponent templates created in the Blueprint editor
- `ComponentClassOverrides` — Class overrides for inherited components

## Key Functions

- `GetDefaultObject()` — Returns the CDO (Class Default Object) with default property values.
- `GetSuperClass()` — Returns the parent UClass.
- `GetGeneratedClassBlueprint()` — Returns the UBlueprint asset that generated this class.
- `GetComponentTemplates()` — Returns the array of component templates.

## Blueprint Graph Structure

The Blueprint visual scripting system uses UEdGraph objects containing UEdGraphNode instances connected by UEdGraphPin links:

- `UEdGraph` — A single graph (event graph, function, macro)
- `UEdGraphNode` — A node in the graph (function call, event, variable get/set, flow control)
- `UEdGraphPin` — Input/output pin on a node (exec pins for flow, data pins for values)
- `UK2Node` — Base class for Blueprint-specific nodes
- `UK2Node_CallFunction` — Node that calls a function
- `UK2Node_Event` — Event node (BeginPlay, Tick, custom events)
- `UK2Node_VariableGet` / `UK2Node_VariableSet` — Variable access nodes
